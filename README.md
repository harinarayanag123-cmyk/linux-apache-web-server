# Linux Apache Static Website Hosting

## Overview
This project demonstrates hosting a static website using Apache HTTP Server on an AWS EC2 instance.

## Tech Stack
- Linux
- Apache HTTP Server
- HTML
- CSS
- AWS EC2

## Implementation
- Installed Apache server
- Developed static website using HTML & CSS
- Deployed website in /var/www/html
- Configured file permissions and symbolic links

## Linux Operations
- Used grep for searching
- Used sed for text processing
- Managed file permissions
- Created symbolic links

## Key Features
- Web server configuration
- Linux command-line operations
- Static website deployment

## Results
- Successfully hosted website on EC2
- Gained hands-on Linux administration experience
